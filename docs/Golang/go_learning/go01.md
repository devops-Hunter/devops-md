## 📋 结构体

---
<Alert type="info">
结构体用法：用来模拟其他语言中的类
</Alert>

### 🔨 定义结构体&初始化结构体の三种方法
```go
func main() {
	// 结构体数据类型，可以存储json这种数据，绑定方法
	// 1、通过var关键字来实例化结构体
	var p1 Person
	p1.Age = 28
	p1.name = "zhangsan"
	fmt.Println(p1)
	fmt.Println(p1.name)

	// 2、new方法实例化一个结构体(返回指针类型)
	var p2 = new(Person)
	p2.Age = 25
	p2.name = "lisi"
	fmt.Println(p2)

	// 3、:= 键值对初始化
	p3 := Person{
		name: "wangwu",
		Age:  35,
	}
	fmt.Println(p3)

}

/*
type: 关键字
Person： 结构体名字
struct：  定义结构体的关键字
*/
// golang中严格区分大小写（json序列化的时候，或者不同包访问是不能访问小写）
type Person struct {
	name string
	Age  int
}
```
